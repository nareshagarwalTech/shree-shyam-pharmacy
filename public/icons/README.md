# PWA Icons Setup

## Quick Setup (Recommended)

1. Go to https://realfavicongenerator.net
2. Upload the `icon.svg` file from this folder
3. Download the generated icons
4. Extract and replace files in this `/public/icons/` folder

## Required Icon Files

Create these PNG files from the SVG:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

Also create:
- /public/favicon.ico (use 32x32 or 48x48)

## Alternative: Use Canva

1. Open Canva.com
2. Create 512x512 design
3. Make a pill icon with emerald green (#10b981) background
4. Download as PNG
5. Use https://www.pwabuilder.com/imageGenerator to create all sizes

## Temporary Solution

The app will work without icons, but won't look professional on home screen.
Use the SVG icon directly for now - modern browsers support it.
